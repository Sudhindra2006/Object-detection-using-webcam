# Object-detection-using-web-camera

This project uses YOLOv4 and OpenCV to perform real-time object detection using a webcam.

## Features
- Real-time object detection
- Webcam video capture
- Bounding boxes and labels
- Uses COCO dataset classes

## Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib
- YOLOv4

## Files Required
- yolov4.weights
- yolov4.cfg
- coco.names
